## Overview
**Survive The Month** is an interactive web project built for the Hackonomics 2026 Hackathon.  
It is designed to simulate financial decision-making and help users understand how budgeting choices impact survival over a monthly cycle.

The goal is to make personal finance concepts more engaging, visual, and decision-driven.

---

## Features
- Monthly budget simulation experience  
- Interactive decision-based gameplay  
- Real-world inspired financial tradeoffs  
- Strategy-based progression system  
- Responsive design for all devices
